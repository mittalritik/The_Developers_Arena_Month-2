# The_Developers_Arena_Month-2

## 🚀 Overview

This repository contains multiple projects focused on:

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Data Visualization
* Insight Generation
* Real-world Problem Solving

Each week focuses on a different domain to build strong analytical skills.

## 📅 Weekly Breakdown

---

### 🛒 Week 1: Supermarket Sales Analysis

**Objective:** Analyze daily sales and customer purchasing behavior.

**Key Tasks:**

* Identify best-selling products
* Analyze daily and weekly sales patterns
* Study customer buying behavior
* Revenue trend analysis

**Tools Used:** Python, Pandas, Matplotlib, Seaborn

---

### 🎓 Week 2: Student Performance Analysis

**Objective:** Evaluate student academic performance.

**Key Tasks:**

* Calculate pass/fail rates
* Subject-wise performance analysis
* Correlation between attendance and performance
* Identify high-performing and weak students

**Outcome:** Insights into factors affecting student success

### 🌦️ Week 3: Weather Data Analysis

* Detection of extreme weather events

**Outcome:** Understanding climate variability and patterns
